# Настройка переменных окружения

## Создание файла .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```bash
# Server configuration
PORT=3000
NODE_ENV=production

# Admin panel credentials (необязательно)
# ADMIN_USERNAME=admin
# ADMIN_PASSWORD=your_secure_password
```

## Как изменить порт

1. Откройте файл `.env`
2. Измените значение `PORT=ваш_порт`
3. Перезапустите контейнер: `docker-compose down && docker-compose up -d`

## Примеры портов

- `PORT=3000` - приложение будет доступно на http://localhost:3000
- `PORT=8080` - приложение будет доступно на http://localhost:8080
- `PORT=80` - приложение будет доступно на http://localhost:80

## Как это работает

- Приложение внутри контейнера всегда использует порт из переменной `PORT`
- Docker Compose пробрасывает указанный порт хоста на порт 3000 внутри контейнера
- Если переменная `PORT` не указана, используется порт 3000 по умолчанию
